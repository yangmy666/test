# 个人人际相处沉淀

> 本文件只记录可以公开分享的"自我观察 + 通用规律"。任何涉及具体他人的细节、原话、关系状态都**不在此文件中**。

## 关于我自己（自我观察）

- 在感到被拒或不被回应时，容易做出冲动决定（例如切断联系），事后大概率会后悔。
  - 改进方向：感到挫败时**先过 24 小时**再做任何回应或动作；不要把"切断联系"当作情绪出口。
- 倾向于按字面意思理解对话，容易错过对方话语里的暗示。
  - 改进方向：当对方反复抛出"假设性问题"，多半是在投射自己的状态或顾虑，多想一步再回。
- 在邀约被拒时倾向于继续硬约。
  - 改进方向：第一次被拒就**降低颗粒度**——把"正式相处"换成"低压短时相遇"，把"几小时"压成"半小时"，把"约会"重新框成"顺路做点小事"。

## 通用沟通规律

1. **承担情绪责任 > 逼对方表态**。踩过雷之后主动认错 + 承担 + 不催促，比解释或反驳有效得多。
2. **低压框架 > 高压邀约**。明确的"短时长、不正式、可控、可中途撤"承诺，比"我想认真见一面"更容易被接受。
3. **节奏比强度重要**。对方松口的当下不要立刻追问具体时间；预留 2~3 天缓冲让对方自己消化。
4. **永远不要把别人的话/身份/状态写到会被外人读到的地方**——这是踩过坑后的硬规则。
